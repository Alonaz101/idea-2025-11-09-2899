# Overview of Features

This repository implements the MVP for a Mood-based Recipe Recommendation system with features derived from Jira issues SCRUM-178 through SCRUM-185.

## Features and Jira Traceability

- **SCRUM-178**: Core mood input interface and recipe recommendation API with rule-based matching and response time guarantee (<2s).
- **SCRUM-179**: User profile management and mood history tracking with secure data storage.
- **SCRUM-180**: Favorites feature with backend APIs and frontend UI components.
- **SCRUM-181**: Social sharing integration with OAuth simulation and privacy controls.
- **SCRUM-182**: NLP mood input processing analyzing free-text inputs and returning detected mood.
- **SCRUM-183**: AI-driven personalized recipe generation microservice stub.
- **SCRUM-184**: Community recipe submission, rating platform with moderation controls and APIs.
- **SCRUM-185**: Multi-language support and internationalization framework for frontend and backend.

## Repository Structure

- `backend/` - Contains FastAPI backend app and modules for each feature.
- `docs/overview.md` - This document summarizing the implemented features.

This codebase is ready for further extension and production deployment.
